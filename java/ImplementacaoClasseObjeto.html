06. Implementação de Classes e Objetos

Como instanciar um objeto em Java

Quando você cria uma classe em Java, ela por si só não executa nada. Para utilizá-la, é necessário criar um objeto a partir dela (instanciação).

1. Criando uma classe simples

Arquivo:

Pessoa.java

Conteúdo:


public class Pessoa {

    String nome;
    int idade;
    String curso;

    void apresentar() {
        System.out.println("Olá, meu nome é " + nome);
        System.out.println("Minha idade é " + idade);
        System.out.println("Meu curso é " + curso);
    }
}

2. Programa principal

Main.java

public class Main {
    public static void main(String[] args) {

        Pessoa p1 = new Pessoa();

        p1.nome = "Carlos";
        p1.idade = 17;
        p1.curso = "Técnico em Informática";

        p1.apresentar();
    }
}

3. Entendendo a instanciação


Pessoa p1 = new Pessoa();
  • Pessoa: tipo
  • p1: variável
  • new Pessoa(): instanciação

4. Acessando o objeto

Atribuição de valores


p1.nome = "Carlos";
p1.idade = 17;
p1.curso = "Técnico em Informática";

Chamada de método


p1.apresentar();

5. Fluxo completo

  1. Criar a classe
  2. Criar o Main
  3. Instanciar objeto
  4. Utilizar objeto
