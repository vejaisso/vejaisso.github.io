Implementação de Classes e Objetos em Java

Aula 06 — Programação Orientada a Objetos

Introdução

Nesta aula, implementamos os conceitos fundamentais de orientação a objetos utilizando Java.

Definição de Classe

Uma classe define atributos e comportamentos de um objeto. 

class Pessoa {
    String nome;
    int idade;
}

Criação de Objetos

Objetos são instâncias de classes. 

Pessoa p = new Pessoa();
p.nome = "João";
