Implementação de Classes e Objetos em Java

Aula 06 — Programação Orientada a Objetos

Introdução

Nesta aula, vamos implementar os conceitos fundamentais de orientação a objetos utilizando a linguagem Java.

Definição de Classe

Uma classe é uma estrutura que define atributos e comportamentos de um objeto. 

class Pessoa {
    String nome;
    int idade;
}

Criação de Objetos

Objetos são instâncias de uma classe. 

Pessoa p = new Pessoa();
p.nome = "João";
